# 引言 {#introduction}

## 为什么写这个知识库？

2022年11月30日，OpenAI发布了ChatGPT。这个看似简单的聊天机器人，改变了整个科技行业的走向。

在随后的几年里，我们见证了：

- **百模大战**：全球数百家公司竞相推出大语言模型
- **开源革命**：LLaMA的发布催生了繁荣的开源AI生态
- **Agent时代**：从聊天机器人到自主智能体的范式转变
- **芯片战争**：NVIDIA成为全球最有价值的公司之一
- **中国崛起**：DeepSeek等中国AI力量的崛起
- **推理突破**：o1/DeepSeek-R1等推理模型的出现
- **MCP协议**：AI工具连接的标准化
