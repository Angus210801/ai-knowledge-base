## 本地运行工具

### Ollama
| 项目 | 信息 |
|------|------|
| **功能** | 一行命令运行本地LLM |
| **特点** | 类似Docker的体验 |
| **支持** | LLaMA, DeepSeek, Qwen等 |
| **平台** | macOS, Linux, Windows |

```bash
ollama run llama3.1
ollama run deepseek-r1
ollama run qwen2.5
```
