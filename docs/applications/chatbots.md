# 对话式AI {#chatbots}

对话式AI是AI最广泛的应用形式，从ChatGPT到各种AI助手。

---

## 主要产品

### ChatGPT
| 项目 | 信息 |
|------|------|
| **公司** | OpenAI |
| **发布** | 2022年11月 |
| **用户** | 2亿+周活 |
| **版本** | 免费版, Plus ($20/月), Pro ($200/月) |

**能力**：
- 文本对话
- 图像理解
- 代码编写
- 文件分析
- 网页搜索
- DALL-E图像生成

## 2025年趋势

1. **Agent能力**：从回答问题到执行任务
2. **多模态交互**：语音、图像、视频
3. **个性化**：记住用户偏好
4. **企业版**：更多企业级功能
5. **竞争加剧**：更多产品涌现
