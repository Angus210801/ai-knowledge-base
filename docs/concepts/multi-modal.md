# 多模态AI {#multi-modal}

多模态AI是指能够理解和生成多种类型内容（文本、图像、视频、音频）的AI系统。

---

## 什么是多模态？

**模态（Modality）**：信息的类型
- **文本**：文字、代码
- **图像**：照片、图表、设计
- **视频**：动态画面
- **音频**：语音、音乐
- **3D**：三维模型

**多模态AI**：能够处理两种或以上模态的AI系统。

---

## 多模态的发展历程

### 阶段一：单模态（2020-2022）
- GPT-3：只能处理文本
- DALL-E：只能生成图像
- Whisper：只能处理语音

### 阶段二：多模态输入（2023）
- GPT-4V：文本+图像输入
- Claude 3：文本+图像输入
- Gemini 1.0：原生多模态训练

### 阶段三：多模态输入输出（2024-2025）
- GPT-4o：文本+图像+音频输入输出
- Gemini 2.0：原生多模态输出
- Claude 4：文本+图像输入

## 原生多模态 vs 拼接多模态

### 拼接多模态（早期方法）
```
文本模型 + 视觉编码器 = 多模态
```
- 各模态独立训练
- 通过适配器连接
- 理解深度有限

### 原生多模态（现代方法）
```
在所有模态上联合训练
```
- 从一开始就在多模态数据上训练
- 模态间理解更深入
- **代表**：Gemini, GPT-4o

## 多模态的应用场景

### 1. 文档理解
- 分析PDF、合同、发票
- 提取表格数据
- 理解图表

### 2. 医疗影像
- X光、CT、MRI分析
- 辅助诊断

### 3. 自动驾驶
- 理解道路场景
- 识别行人、车辆

### 4. 创意设计
- AI辅助设计
- 图像编辑
- 视频制作

### 5. 教育
- 图片内容讲解
- 视频内容总结

## 2025年多模态趋势

1. **统一模型**：一个模型处理所有模态
2. **实时交互**：实时语音、视频对话
3. **视频理解**：理解长视频内容
4. **3D生成**：文本到3D模型
5. **世界模型**：理解物理世界的AI
