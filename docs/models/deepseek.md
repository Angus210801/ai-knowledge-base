# DeepSeek {#deepseek}

深度求索（DeepSeek）是中国最具突破性的AI公司，以极低的成本训练出世界级模型，震惊全球AI行业。

---

## 公司概况

| 项目 | 信息 |
|------|------|
| **成立时间** | 2023年7月 |
| **创始人** | 梁文锋（Liang Wenfeng） |
| **母公司** | 幻方量化（High-Flyer，量化对冲基金） |
| **总部** | 中国杭州 |
| **核心理念** | 用更少的资源做更好的AI |
| **融资** | 幻方量化自有资金，未大规模外部融资 |

## 模型演进

### DeepSeek V1（2024年1月）
- **参数**：67B
- **特点**：首个公开发布的模型
- **意义**：开始引起关注

### DeepSeek V2（2024年5月）
- **参数**：236B（21B激活，MoE架构）
- **特点**：
  - DeepSeekMoE架构
  - Multi-head Latent Attention (MLA)
  - 成本极低（API定价远低于竞争对手）
- **意义**：证明了高效架构的潜力

### DeepSeek V3（2024年12月26日）
- **参数**：671B（37B激活，MoE架构）
- **训练成本**：仅557万美元（GPT-4的1/10）
- **训练数据**：14.8万亿token
- **特点**：
  - 性能媲美GPT-4o和Claude 3.5 Sonnet
  - 训练成本极低
  - 开源（MIT许可）
- **意义**：震惊全球AI行业，证明中国AI实力

### DeepSeek R1（2025年1月20日）
- **参数**：671B（与V3相同基础架构）
- **关键突破**：
  - 开源推理模型，性能媲美OpenAI o1
  - 在AIME 2024测试中得分79.8%（与o1相当）
  - 训练成本极低
  - 使用强化学习训练推理能力
- **影响**：
  - 导致NVIDIA股价一天内暴跌17%
  - 证明开源推理模型的可行性
  - 全球媒体广泛报道
- **意义**：开源推理模型的里程碑

### DeepSeek R1 Zero
- 纯RL训练，无SFT
- 展示了"涌现"推理能力的可能

### DeepSeek R1蒸馏版本
- 1.5B / 7B / 8B / 14B / 32B / 70B
- 小模型也能获得强推理能力
- 可在消费级硬件上运行

### DeepSeek R2（2025年6月）
- **推理能力**：在AIME 2025、MATH-500等测试中超越R1
- **多模态**：支持图像输入和推理
- **效率**：训练效率进一步优化
- **开源**：继续MIT许可开源
- **意义**：证明DeepSeek的创新能力持续领先

### DeepSeek R1蒸馏版（持续更新）
- **版本**：1.5B/7B/8B/14B/32B/70B
- **特点**：小模型获得强推理能力
- **应用**：可在消费级硬件运行（RTX 4090等）
- **影响**：让推理能力民主化

## 核心技术突破

### DeepSeekMoE
- 改进的MoE架构
- 更细粒度的专家划分
- 更高效的专家路由

### Multi-head Latent Attention (MLA)
- 创新的注意力机制
- 大幅降低KV cache内存占用
- 推理效率提升数倍

### 低成本训练
- V3训练成本仅557万美元
- 对比：GPT-4训练成本估计1亿美元+
- **关键因素**：
  - 高效的MoE架构
  - 创新的训练方法
  - 优秀的工程优化

### 强化学习训练推理
- R1使用纯RL训练推理能力
- 不依赖大量人工标注的推理数据
- 推理能力"涌现"

## 为什么DeepSeek如此重要？

1. **打破垄断**：证明不需要数十亿美元也能训练世界级模型
2. **开源力量**：开源模型首次匹敌最强闭源模型
3. **中国AI崛起**：证明中国在AI领域有真正的创新能力
4. **成本革命**：改变了"更大=更好"的行业叙事
5. **技术民主化**：让更多人能使用和改进顶级AI

---

## 行业影响

### 对NVIDIA的冲击
- DeepSeek R1发布后NVIDIA市值蒸发近6000亿美元
- 引发"AI是否需要这么多GPU"的讨论
- 但实际上NVIDIA需求并未减少

### 对行业的启示
- 高效训练比暴力扩展更重要
- 开源可以匹敌闭源
- 创新不一定需要巨额投资
