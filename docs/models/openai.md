## GPT系列模型演进

### GPT-3.5（2022年11月）
- **参数**：175B
- **上下文**：4K token
- **意义**：ChatGPT的基础模型，首次让大众体验到AI对话

### GPT-4（2023年3月14日）
- **参数**：未公开（传闻1.8T MoE）
- **上下文**：8K / 32K
- **关键突破**：
  - 首个多模态GPT（支持图像输入）
  - 推理能力大幅提升
  - 通过律师资格考试（前10%）
- **影响**：树立了大模型能力的新标杆

### GPT-4 Turbo（2023年11月）
- **上下文**：128K
- **知识截止**：2023年4月
- **定价**：比GPT-4便宜3倍
- **新增**：JSON Mode、函数调用增强

### GPT-4o（2024年5月13日）
- **"o"代表"omni"（全能）
- **关键突破**：
  - 原生多模态（文本+图像+音频统一模型）
  - 实时语音对话（延迟232ms）
  - 成本降低50%
- **意义**：多模态AI的里程碑

### GPT-4o mini（2024年7月）
- 替代GPT-3.5 Turbo
- 成本极低（输入$0.15/M tokens）
- 性能远超GPT-3.5

### o1（2024年9月12日）
- **关键突破**：
  - 首个"推理模型"
  - 使用"思维链"深度推理
  - 在AIME数学竞赛中得分83.3%（GPT-4仅12%）
  - 在编程竞赛中超越人类专家
- **定价**：$15/M input, $60/M output
- **意义**：开创了"推理模型"这一新品类

### o1-mini（2024年9月）
- 轻量级推理模型
- 编程和数学能力强，但通用知识较少

### o3（2024年12月）
- 推理能力进一步提升
- ARC-AGI测试得分87.5%

### o3-mini（2025年1月）
- 性价比极高的推理模型

### GPT-4.5（2025年3月）
- **定位**：OpenAI称之为"最后的非推理模型"
- **上下文**：256K token
- **特点**：
  - 知识面更广，情感理解更强
  - 世界知识比GPT-4o多
  - 但推理深度不如o系列
- **定价**：$75/M input, $150/M output（研究预览期间）

### o3（2025年1月31日公开发布）
- **ARC-AGI**：得分87.5%
- **编程**：在SWE-bench上表现突出
- **定价**：$10/M input, $40/M output
- **特点**：强大的推理能力，可调用工具

### o3-mini（2025年1月）
- **定位**：o3的轻量版本
- **定价**：$1.1/M input, $4.4/M output
- **特点**：极高的推理性价比

### o4-mini（2025年5月）
- **定位**：新一代轻量推理模型
- **特点**：在推理任务上表现优异，成本更低
- **支持**：图像推理、工具调用
- **定价**：$1.1/M input, $4.4/M output

### GPT-5（2025年中）
- **上下文**：256K token
- **关键突破**：
  - 统一了GPT系列和o系列（自动路由快速/深度推理）
  - 综合能力全面提升
  - 原生多模态+深度推理
  - 可以自动判断是否需要深度思考
- **性能**：在编程、数学、推理等全面领先
- **定价**：$2/M input, $8/M output
- **意义**：OpenAI首个统一推理与通用能力的模型

### o3-pro（2025年6月10日）
- **定位**：OpenAI最强推理模型
- **性能**：
  - 数学推理：AIME 2024接近满分
  - 编程：SWE-bench行业领先
  - 科学推理：GPQA Diamond领先
- **定价**：$20/M input, $80/M output
- **特点**：
  - 回答前进行极其深度的思考
  - 准确率极高
  - 适合最困难的推理任务
- **限制**：速度较慢，成本极高

## 核心技术特点

1. **Scaling Law**：OpenAI坚信"大力出奇迹"，持续增加模型规模
2. **RLHF**：人类反馈强化学习，让模型更符合人类偏好
3. **多模态融合**：从文本到图像到音频的统一架构
4. **推理链**：o系列模型的"思考再回答"范式
