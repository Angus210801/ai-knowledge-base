# AI芯片全景 {#chips}

除了NVIDIA，全球还有多家公司在AI芯片领域竞争，形成了多元化的AI硬件生态。

---

## AI芯片分类

| 类型 | 代表 | 特点 |
|------|------|------|
| **GPU** | NVIDIA, AMD | 通用并行计算 |
| **TPU** | Google | 张量计算专用 |
| **NPU** | Intel, Apple | 神经网络专用 |
| **FPGA** | Xilinx (AMD) | 可编程逻辑 |
| **ASIC** | Cerebras, Groq | 定制化芯片 |

---

## 主要玩家

### AMD

#### MI300X（2023年）
| 参数 | 规格 |
|------|------|
| 显存 | 192GB HBM3 |
| 带宽 | 5.3 TB/s |
| 价格 | 约$10,000-15,000 |

**优势**：
- 内存比H100更大
- 性价比高
- ROCm生态改善

**挑战**：
- 软件生态不如CUDA
- 框架支持不够完善

#### MI350（2025年预期）
- 下一代AI芯片
- 对标B200

---

### Google TPU

#### TPU v4（2022年）
- Google自用
- 支持Gemini训练

#### TPU v5e（2023年）
- 针对推理优化
- 成本更低

#### TPU v5p（2023年）
- 针对训练优化
- 性能更强

#### Trillium / TPU v6e（2024年）
- 最新一代
- 性能提升4倍以上

**特点**：
- Google自用，不对外销售
- 通过Google Cloud使用
- 成本比NVIDIA低

---

### 华为昇腾

#### Ascend 910B（2023年）
| 参数 | 规格 |
|------|------|
| AI性能 | 320 TFLOPS (FP16) |
| 显存 | 64GB HBM2e |
| 特点 | 中国最强AI芯片 |

#### Ascend 910C（2024年）
- 性能提升
- 对标A100

**挑战**：
- 美国制裁限制先进制程
- 软件生态（CANN）不如CUDA
- 产能受限

**意义**：
- 中国AI发展的关键支撑
- DeepSeek等公司使用

---

### Intel

#### Gaudi 3（2024年）
- AI加速器
- 性价比不错
- 市场份额小

#### Lunar Lake / Arrow Lake
- 内置NPU
- 端侧AI推理

---

### Cerebras

#### WSE-3（Wafer Scale Engine）
| 参数 | 规格 |
|------|------|
| 晶体管 | 4万亿 |
| 核心 | 900,000 |
| 内存 | 44GB SRAM |
| 特点 | 整个晶圆一个芯片 |

**特点**：
- 世界最大芯片
- 适合超大模型训练
- 无需多GPU互联

---

### Groq

#### LPU（Language Processing Unit）
- 专为推理设计
- 极快的推理速度
- 确定性延迟

**特点**：
- 推理速度比GPU快10倍+
- 适合实时应用
- 成本较低

---

### Graphcore

#### IPU（Intelligence Processing Unit）
- 处理器内大内存
- 适合稀疏计算
- 英国公司

---

### Apple

#### M系列芯片
- 内置Neural Engine
- 端侧AI推理
- Apple Intelligence

## 2025年芯片趋势

1. **内存增长**：更大HBM应对更大模型
2. **推理专用**：更多推理优化芯片
3. **能效提升**：更低功耗
4. **中国追赶**：华为昇腾持续进步
5. **自研芯片**：大厂自研芯片增加
6. **端侧AI**：手机/PC内置AI芯片
